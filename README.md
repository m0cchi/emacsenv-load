# emacsenv-load
load argsfile

# usage
```bash
$ echo 'install emacs-25.1 --without-x' > build-option
$ emacsenv load build-option
;; ...
;; ...
;; ...
;; install emacs-25.1!!!
```

# usage
``` bash
$ VERSION_NAME='foobar' emacsenv load $OTHER_USER_DIR/.emacsenv/versions/emacs-25.1/build-option.emacsenv
;; same option
```
